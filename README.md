# Pay Rate Calculator

A single-page tool to visualise how a GST surcharge affects the artist/studio pay split — showing the difference between what the client pays and what the artist actually takes home under each system.

## What it does

Enter either a base rate or a client charge, set the artist's percentage, and the calculator shows a full side-by-side breakdown:

- **Old system** — the agreed split of the base rate
- **New system** — what happens when a 10% surcharge is added to the quoted rate, but the artist's pay stays pegged to the original base amount
- **Comparison summary** — the artist's actual percentage of the client charge, and the additional studio revenue generated by the surcharge

## Inputs

- **Base Rate** — the artist's quoted price before any surcharge. Updating this field recalculates the client charge automatically.
- **Client Charge** — the amount the client is quoted (base rate + 10%). Updating this field works backwards to recalculate the base rate automatically. Both fields are linked and editable.
- **Artist %** — the artist's agreed share of the base rate. The studio percentage updates automatically to make up 100%.
